Worldwide Weather Watcher

This project is a school project created by a team of four students, including myself. It focuses on developing a weather station that can collect and record meteorological data. The system is designed to operate in different modes, including standard, configuration, maintenance, and energy-saving, allowing users to customize its functionality based on their needs. Data is acquired at regular intervals and stored on an SD card for later analysis.

The weather station includes various sensors to measure parameters like temperature, humidity, pressure, and light intensity. It also features an LED color-coded system for easy status monitoring and error detection. Users can modify system parameters through the configuration mode, ensuring flexibility in data acquisition. Overall, the project demonstrates practical applications of embedded systems and sensor integration, offering a hands-on learning experience in electronics and programming.
